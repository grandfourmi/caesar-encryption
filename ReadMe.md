Программа реализована в рамках обучения языку программирования JAVA. Является итоговым проектом
модуля 1 на портале JAVARUSH.com.
Написана на основе программы по расшифровки кода морзе Sergey Proshchaev (sproshchaev GIT HUB).
Программа сдвигает символы в тексте (только буквы, символы и пробелы игнорируются и русская буква "Ё")
на заданную величину для кодировки и раскодировки текста. Осуществляет1 проверку на наличие входного файла.
Имеет несколько режимов работы:

1. Кодирование.
   Вводите адрес где читать, затем адрес откуда, и величину сдвига. После программа запишет новый файл по вашему адресу.
2. Раскодирование.
   Аналогично предыдущему. Даете адрес для чтения, адрес для записи и ключ равный тому, что был указан при
   кодировании.  
   ВНИМАНИЕ при раскодирование смешанных текстов возможны ошибки при коде выше 25 тк программа будет "крутить" латиницу
   на меньший шаг step = step % 26;
   Введенные отрицательные числа будут взяты по модулю. Для моно язычного текста значения кодировки выше 32 (кириллица)
   и выше 26 (латынь) будут так же взяты от остатка деления step = step % 32/ step = step % 26 соответственно.
3. Взлом путем перебора.
   Программа запрашивает путь чтения/записи, а потом делает сдвиг текста на 1 вперед и дает превью. Если пользователя
   устраивает результат, то записывает в файл весь текст.
4. Раскодирование статистическим анализом работает с текстом моно алфавитным, о чем спрашивает при выборе пункта
   меню.
   От выбора пользователя будет дифференцироваться символы которые будут раскодироваться, но это всега символы
   алфавита
   кириллица или латынь. Весь класс написан под алгоритм придуманным ИИ код оригинальный.
5. При вызове команды на выход программа закрывает поток ввода и завершается.
